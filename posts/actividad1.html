Actividad 1: Maquetación HTML5/CSS3

Publicado el 15 de Octubre de 2024 • Tiempo de lectura: 5 minutos
HTML5 CSS3 Responsive Maquetación

Objetivo de la actividad

El objetivo principal de esta primera actividad fue crear una página web completamente responsive utilizando HTML5 semántico y CSS3, aplicando las mejores prácticas de desarrollo web moderno.

Tecnologías utilizadas

  • HTML5 con estructura semántica
  • CSS3 con Flexbox y Grid
  • Media Queries para diseño responsive
  • Variables CSS para mantener consistencia

Resultados obtenidos

Se logró crear una interfaz limpia y profesional que se adapta correctamente a diferentes dispositivos y tamaños de pantalla. La página incluye:

  • Header con navegación
  • Sección hero con información principal
  • Grid de posts o artículos
  • Footer con información adicional

Aprendizajes clave

Esta actividad me permitió reforzar conceptos fundamentales de maquetación web y entender la importancia de la semántica en HTML5 para la accesibilidad y el SEO.
